# Vue.js

## Started

```
npm install -g @vue/cli
```

으로 pc에 기반 설치 (최초)

or 설치여부  확인

```
vue --version
```



vue 시작 

```
vue create [app이름] >> 버전 설정
npm i -D sass sass-loader@10
npm i bootstrap
npm i axios
vue add router
vue add vuex
```



App.vue에 추가

```
...
<style lang="scss">
@import 'bootstrap/scss/bootstrap'
</style>
```
