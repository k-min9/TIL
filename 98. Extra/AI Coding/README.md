# AI Coding

## 종류

Code to Code : 사람이 코드를 입력, AI가 코드를 출력
Text to Code : 사람이 자연어 입력, AI가 코드 출력
Code to Text : 사람이 코드 입력, AI가 자연어 출력

## 기술 동향

- 상용도구
  - 빅테크 기업 : ChatGPT, Copilot, Tabnine, CodeWhisperer, Google Research
  - 오픈소스 커뮤니티 : chatGPT 출몰 이후, 논문이나 기업이 폐쇄적 분위기로 변하며 하나 둘 출몰
- 문제점
  - SI 프로젝트에 적용하는 응용 프레임워크 기반의 코드를 생성해주기는 힘듬
    - 기술적이기보다는 비지니스 & DB 중심의 알고리즘이기 때문
- 해결
  - 코드 학습이 아닌, 프롬프트 방식으로 해결

## Prompt 학습

- 페르소나(역할) 부여
- Meta-Prompt : Prompt를 Prompt
